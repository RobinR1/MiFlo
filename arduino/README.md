# Arduino code voor de MiFlo

Deze code is geschreven voor een [arduino feather huzzah](https://learn.adafruit.com/adafruit-feather-huzzah-esp8266) in combinatie met een [gameduino touchscreen](https://www.watterott.com/en/Gameduino-3). 

Je hebt ook een RTC module nodig, bijvoorbeeld dit [rtc-breackoutboard](https://www.adafruit.com/product/3295)
